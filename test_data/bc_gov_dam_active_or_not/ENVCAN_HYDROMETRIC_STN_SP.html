Hydrometric Stations - Active and Discontinued

637 views (11 recent)

Published by the Ministry of Environment and Climate Change Strategy - Knowledge Management
Licensed under Open Government Licence - British Columbia

BC active and discontinued hydrometric stations (surface water level and flow data) that are part of the provincial hydrometric network managed under a national program jointly administered under a federal-provincial cost-sharing agreement with Environment and Climate Change Canada (ECCC).

Dataset Extent

Latitude: 48.0° to 60.0°
Longitude: -139.5° to -113.5°
View in iMapBC

Additional Information

Purpose

To identify existence, status and locations of active & discontinued hydrometric stations
Data Quality
  • Active and discontinued hydrometric stations are differentiated using the STATION_OPERATING_STATUS attribute field.
  • For active stations, STATION_OPERATING_STATUS = 'ACTIVE' or STATION_OPERATING_STATUS = ‘ACTIVE-REALTIME'
  • For discontinued stations, STATION_OPERATING_STATUS = 'DISCONTINUED'

Object Description

Object Name: WHSE_ENVIRONMENTAL_MONITORING.ENVCAN_HYDROMETRIC_STN_SP

Short Name: ENVCAN_HYD
Comments: Hydrometric stations (HYDROMETRIC STN SP ) are active and discontinued hydrometric stations in BC. The Water Survey of Canada maintains hydrometric (streamflow and water level) records for 2400 hydrometric stations in British Columbia and Yukon for which over 500 are active at present. This is not a multipart feature class

Column Name Short Name Data Type Data Precision Comments
HYDROMETRIC_STATION_ID HDRMTRCSTT NUMBER 18
STATION_NUMBER STATION_NO VARCHAR2 10 STATION NUMBER is the unique identifier for a Hydrometric Station.
FEATURE_CODE FEAT_CODE VARCHAR2 10 FEATURE CODE is a standard alphanumeric code to identify the type of feature represented by the spatial data. A feature code is most importantly a means of linking a feature to its name and definition. For example, the code GB15300120 on a digital geographic feature links it to the name "Lake - Dry" with the definition "A lake bed from which all water has drained or evaporated." The feature code does NOT mark when it was digitized, what dataset it belongs to, how accurate it is, what it should look like when plotted, or who is responsible for updating it. It only says what it represents in the real world. It also doesn't even matter how the lake is represented. If it is a very small lake, it may be stored as a point feature. If it is large enough to have a shape at the scale of data capture, it may be stored as an outline, or a closed polygon. The same feature code still links it to the same definition.
STATION_NAME STATN_NAME VARCHAR2 60 STATION_NAME is the Hydrometric Station Name. e.g., "LINDEMAN CREEK NEAR BENNETT", "ATLIN LAKE AT ATLIN".
FLOW_TYPE FLOW_TYPE VARCHAR2 16 FLOW_TYPE represents Natural or Regulated flow. i.e., "NATURAL" or "REGULATED".
WATERSHED_GROUP_CODE WTR_GRP_CD VARCHAR2 4 WATERSHED_GROUP_CODE refers to Watershed Atlas Watershed Group. WATERSHED GROUP CODE will have null values for non-spatially verified stations.
WATERSHED_ID WATRSHD_ID VARCHAR2 15 WATERSHED_ID is used for Watershed Atlas Identification. WATERSHED ID will have null values for non-spatially verified stations.
STREAM_ORDER STREAM_ORD NUMBER 4 STREAM_ORDER stores the stream order as a numeric value. The stream order is based on 1:50000 scale mapping.
ARCHIVE_URL ARCHIV_URL VARCHAR2 200 ARCHIVE_URL is a URL for Environment and Climate Change Canada web site showing archival flow information for a specific station.
REALTIME_URL REALTM_URL VARCHAR2 200 REALTIME_URL is a URL for Environment and Climate Change Canada web site showing real time water level data for a specific station.
STATION_OPERATING_STATUS STN_OP_STA VARCHAR2 15 STATION_OPERATING_STATUS represents the Hydrometric Station Status. i.e., "ACTIVE”, “ACTIVE-REALTIME”, OR “DISCONTINUED”.
CAPTURE_SCALE CAPTR_SCAL VARCHAR2 10 CAPTURE_SCALE is the scale at which a station point was mapped. e.g., 1:50,000.
START_DATE START_DATE DATE 7 START_DATE is the earliest measurement date of approved hydrometric data for a station.
END_DATE END_DATE DATE 7 END_DATE is the most recent measurement date of approved hydrometric data for a station.
GEOMETRY GEOMETRY SDO_GEOMETRY 1 GEOMETRY is the column used to reference the spatial coordinates defining the feature.
OBJECTID OBJECTID NUMBER 38 OBJECTID is a required attribute of feature classes and object classes in a geodatabase.
SE_ANNO_CAD_DATA BLOB 4000 SE_ANNO_CAD_DATA is a binary column used by spatial tools to store annotation, curve features and CAD data when using the SDO_GEOMETRY storage data type.
